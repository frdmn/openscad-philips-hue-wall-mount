This custom wall mount is designed mount to a Philips Hue dimmer switch on a [Gira Standard 55](https://www.gira.de/schalterprogramme/standard55.html) (EU standard) wall flip switch without the need of a special mount or frame. The Hue switch is fixed by a magnet so it holds nice and properly to the mount. 

## Usage

Here's a short explanation how to use `openscad-philips-hue-wall-mount`:

* Download `.scad` or `.stl` file
* Slice object
* Print using your 3D printer
